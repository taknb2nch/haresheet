# haresheet [![Go Reference](https://pkg.go.dev/badge/github.com/taknb2nch/haresheet.svg)](https://pkg.go.dev/github.com/taknb2nch/haresheet)

**haresheet** is a developer-friendly wrapper for the Google Sheets API v4 in Go. It features **ID-based operations** (robust against sheet renaming) and intuitive **(row, col) coordinates**, simplifying the "Read -> Process -> Write" workflow.

## Installation

```bash
go get https://github.com/taknb2nch/haresheet
```
